金陵的夜和鸭子

发表于 2019-11-28 | 分类于 阿九的世界 |
美得很，都美得很！

翻翻记事本，这两年去南京的次数已经有两位数了。
吃喝玩乐是有不少的，但占据大部分印象的，还是趁着夜色在城里闲逛，避开了熙熙攘攘的人群，好像跟这座城市能更贴近些。
城里四股、两股的街道都算不上宽阔，但足够我们在深夜里恣意游荡。有时候热络的聊着各自感兴趣的，有时候却也会大声激烈的争执些什么。路边的梧桐树就那么安安静静的立在那儿，安安静静的听着我们，不发一言，偶尔一阵风过，沙沙的声音好像是在附和，又好像是在轻笑。
夫子庙的小巷隐没在城市中，没了白日的繁华，秦淮河静静的歇着；新街口仍是比别处更亮堂，大幅广告显示着古城international的一面；玄武湖和紫峰大厦互相映衬着，城门口卖烤玉米的阿姨则会拯救你有点空落落的胃；解放路缘着几家好吃的涮肉店，热闹劲儿不输白天；灵谷寺在萤火虫的季节会堵车到深夜，逃出人群后意外走进了植物园路，黑漆漆的确实有些刺激；百家湖1912看不到“bar区”的声色犬马，取而代之的是充满温馨的家庭感；还有很多，还有很多……

如果说武汉是个接地气的糙汉子，南京则更像天生淘气的富家小姐，在大人面前立着端着成熟懂事，私下里却是娇憨可爱的小丫头。

噢，对了~这个小丫头和我说，她家厨子做的鸭子可好吃啦！
相比于最有名的盐水鸭，我更偏爱烤鸭一些，大约和我有些火爆的性子有关。南京烤鸭的吃法没有北京烤鸭那么讲究和复杂，鸭子就着卤汁，姑娘也能吃下三两米饭。热腾腾的烤鸭，沾着美味的卤汁，一口咬下去，那香味能直接窜到天灵盖。但也不能餐餐吃，顿顿吃，好东西就得欠着吃，欠着吃更香、更有味！没事儿在脑子里回味回味，那美味似乎又多了三分。

来了南京这么多次，也就能写出不到一千字的少少感受，想来是看得还不够多，走得还不够远，嗯，又多了一些继续来南京的理由呢！

2019.11.28 写于武汉
